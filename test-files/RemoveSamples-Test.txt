RemoveSamples Test File
======================

This is a test download for the RemoveSamples-NZBGet extension.

Instructions:
1. Enable Debug: Yes in RemoveSamples settings
2. Enable Test Mode: Yes in RemoveSamples settings  
3. Save changes and restart NZBGet
4. Add this file to NZBGet via Downloads → Add → Choose Files
5. Set category to one that uses RemoveSamples
6. Check Settings → Logging → Messages for test results

Visit: https://github.com/Anunnaki-Astronaut/RemoveSamples-NZBGet
